"Perl + Mojolicious" Web application development with Sakura VPS No. 8 Installing missing Perl modules

Now for the next task, Perl 5.10.1 of CentOS 6 does not include some of the standard Perl modules. This seems to be managed as a separate package by rpm, so let's install it.

The package name is perl-code. Work as the root user. 

#

Make sure that the line starts with # and you are the root user.

RPM is a library management mechanism on CentOS

RPM is an abbreviation for RPM Package Manager, which is a mechanism for managing libraries on CentOS. RPM-managed packages can be installed from the Internet using the command yum . 

You can install RPM managed packages from the internet with the yum command

Installing Perl's Missing Core Modules

Use the yum command to install the missing core module of Perl. You can install RPM packages from the internet with the subcommand install. The -y option is an option that automatically answers "yes" to all commands. 

yum -y install perl-core

This command can be executed manually, but it can also be pasted from the remote console. You can use it because there is a paste button to the right of the part where the IP address is written on the title bar of the remote console. 

The remote console has a paste button, so use it effectively.
