IP address

If you don't understand the basic terms, it will be difficult to study. An IP address is like a number that identifies your computer. Well, properly, the IP address is assigned to the network card that is plugged into the computer.

A NIC (network card) is inserted in the server, and an IP address is assigned to it. If you specify 49.212.148.211 as the IP address, it will follow the router and reach that IP address.

You can reach anywhere in the world. It's amazing to think carefully that it can be identified and accessed.

Hostname

The initial setting is a name that seems to be automatically assigned. www11197ue.sakura.ne.jp. Rest assured that there are of course ways to change this.

The host name has the same meaning as the domain name here. A domain name is a name that identifies it and has the same role as an IP address, but since the IP address is a number, it is not a character string.

The domain name is linked to the IP address, and basically there is a one-to-one correspondence. The DNS server will solve this. In other words, if you access it at www11197ue.sakura.ne.jp, the DNS server will tell you the corresponding IP address.

And if you know the IP address, you can reach the server.
